Windows Flutter 开发环境搭建

2018-07-01
cekiasoo

一、准备

首先要下载 Git， 下面提供了 下载地址和 下载链接   
Git win 下载页面     Git win 64 位 2.18.0 下载链接     Git win 32 位 2.18.0 下载链接

二、获取 Flutter

打开 Git Bash，


切换到合适的位置，输入 git clone -b beta https://github.com/flutter/flutter.git 按回车


把 Flutter 下载下来，下载完后接下来把 flutter 里的 bin 目录添加到 path 路径中去，



然后打开 cmd 窗口，输入 flutter doctor


出现这样的画面是正在下载 Dart SDK


下载完会自解压 Dart SDK 等一系列操作，耐心等待一段时间


三、获取 Android Studio

如果已有 Android Studio 跳至 四 安装插件 ，如果还没有 Android Studio 要先下载 Android Studio
Android Studio 下载页面
Android Studio 64位 3.1.3 安装版下载链接
Android Studio 64位 3.1.3 解压版下载链接
Android Studio 32位 3.1.3 解压版下载链接

打开 Android Studio, 如果是第一次用 Android Studio, 还要有 Android SDK，在 AS 下载或自己电脑有 Andriod SDK 的话也可以指定


四、安装 Flutter 所需的插件

进入 Android Studio 主界面，点击 Configure -> Plugins，安装插件


接着点击 Browse repositories


搜索 Dart ，选择 Dart LANGUAGES 那个，点击右边的原谅色的 Install 按钮安装，因为 Flutter 是用 Dart 语言写的，所以要安装这个插件，


还要再安装个 Flutter 插件


这两个插件安装完后点击 Restart Android Studio


返回主界面时就会有个弹框询问是否重启激活插件，点击 Restart


五、创建项目

重启后，看到没 ? 比之前多了个 Start a new Flutter project 这一行，激动吧


点击 Start a new Flutter project，建一个 Flutter 项目吧，选择 Flutter Application, 点 Next


来个 hello_flutter 吧，虽然很土，但很经典，这里的 Project name 只能是小写字母和下划线组成，再 Next


域名和包名要修改的就修改，默认也可以，Finish


建起来了，建起来了，再次激动一下


手机连上电脑 或 创建个模拟器运行一下吧，在控制台用 flutter devices 命令可以查看连上的设备


点击原谅色的三角形按钮


忙活了大半天终于运行起来了


至此，环境搭建完毕。
